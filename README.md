# 如何参与
---------
将本书fork/pull到本地  
新建一个分支，然后联系我认领要翻译的章节  
使用MarkDown编辑器，md语法格式进行文档翻译及排版工作  
完成之后Pull Request该分支  
如没问题的话，我会合并到主分支  
如果不太明白md的排版，可以吧翻译好的内容发送给我，我代为排版并提交  
如有其它问题，欢迎发送issue，谢谢~
